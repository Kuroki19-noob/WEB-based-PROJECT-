1. Prerequisites
- Node.js (v18+)
- MySQL Server (locally running)

2. Environment Configuration
Create a `.env` file in the root directory and add:
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=lcc
DB_NAME=lagmandb
PORT=8000
JWT_SECRET=your_super_secret_key_here

3. Database Setup
// Required for admin dashboard
CREATE TABLE accounttbl (
    accountID INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(20) NOT NULL,
    email VARCHAR(30) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    role VARCHAR(20) DEFAULT 'user'
);

// To access Admin Dashboard, manually update a user's role to 'admin' in MySQL:
// UPDATE accounttbl SET role = 'admin' WHERE username = 'your_username';

// Required for reservation page
CREATE TABLE reservationttbl (
    id INT AUTO_INCREMENT PRIMARY KEY,
    DateD DATE NOT NULL,
    TableNum INT NOT NULL,
    email VARCHAR(255) NOT NULL,
    number VARCHAR(20) NOT NULL
);

4. Installation & Running
1. Run `npm install`.
2. Start the server using `npm run dev` or `npm start`.
3. Access at http://localhost:8000.

5. Authentication & Security
- The system uses JWT (JSON Web Tokens) for security.
- Sensitive routes (View All Accounts, View Reservations) are protected and require admin role.
- API endpoints follow RESTful conventions: /api/v1/accounts and /api/v1/reservations.
